Embedded systems · Open-source hardware

Frontier TrailCam

A compact, portable timelapse camera built on ESP32-CAM firmware with two startup modes: configuration over a built-in web server (live stream, camera tuning, and capture schedule controls) or autonomous photo capture using saved settings. Between captures, the system enters deep sleep while a DS3231 RTC tracks wake time for low power operation. Images save to a FAT32 microSD card, and the unit runs on dual 18650 batteries for field deployment.
